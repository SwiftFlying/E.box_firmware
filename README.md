# E.box firmware

Firmware for [E.box](http://www.nioonmo.com) microcontroller  

## Getting Started

* Install Arduino IDE  
* From the "Boards Manager" install the package called: "Arduino SAMD Boards (32-bit ARM Cortex-M0+)"  
* Choose the board "Arduino/Genuino Zero (Native USB Port)"
